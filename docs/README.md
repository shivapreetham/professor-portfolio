## Key Technologies

- **Frontend**: Next.js 15, React 19, Tailwind CSS, DaisyUI
- **Backend**: Next.js API Routes, PostgreSQL, Drizzle ORM
- **Authentication**: JWT tokens, bcrypt hashing
- **Hosting**: Vercel-ready, NeonDB compatible
